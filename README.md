## 快速开始

1. **获取 OpenFi 产品**  

   - 访问 [OpenFi 官网](https://www.openfi.one/) 了解并获取产品。

   - 官方淘宝： [OpenFi 淘宝旗舰店](https://openfi.taobao.one) 

2. **下载使用手册**  

   - 访问本仓库的 [使用手册](./docs/manual/) 目录。
   - 下载最新版本的操作说明。

3. **实践操作**  

   - 按照仓库文档进行配置。
